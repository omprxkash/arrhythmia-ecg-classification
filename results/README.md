## Reproducing Results

```bash
# Step 1: download and preprocess data
python src/preprocess.py --data-dir data/mitdb --save-dir data

# Step 2: train all models
python src/train.py --model cnn        --epochs 50 --save results/baseline
python src/train.py --model resnet     --epochs 50 --save results/baseline
python src/train.py --model lstm       --epochs 50 --save results/baseline
python src/train.py --model transformer --epochs 50 --save results/baseline

# Step 3: generate all plots
python results/generate_results.py
```
